# Shopee/Lazada 卖家工具

## 项目定位

面向东南亚电商卖家的小型 AI 工具集合。单人开发，MVP 优先，快速验证市场。

## 目标客户

Shopee/Lazada 卖家，覆盖印尼、泰国、越南、马来西亚、菲律宾市场。

## 技术栈

- Python 3.12+ + FastAPI
- LLM（DeepSeek API / OpenAI API）
- 前端尽量简单（HTMX + Tailwind 或 Chrome 插件）
- 需要爬虫时用 Playwright

## 定价模式

月订阅 $5-20/月，可叠加功能。

## 当前阶段

Brainstorming — 确定第一个产品的方向。

## 约束

- 每个工具应在 3-7 天内完成 MVP
- 优先利用现有技术能力（LLM、爬虫、PDF 生成）
- 以月订阅为目标设计变现模式
- 先做最小可卖版本，不做平台级产品
